Abstract
In this paper, we present an adaptive link position tracking controller for robot manipulators which achieves controller/update law
modularity. Speci"cally, the proposed torque input control law is designed to (i) achieve input-to-state stability with respect to the
parameter estimation error, and (ii) guarantee asymptotic link position tracking and boundedness of all closed-loop signals for any
adaptive update law that satis "es some generic properties. Simulation and experimental results are included to illustrate the
advantages of the proposed adaptive control law. Speci"cally, the proposed controller with a least-squares estimator is compared to
a well-known, gradient update-based, adaptive controller. The results indicate an improvement in the tracking performance with the
proposed controller. ( 1999 Elsevier Science Ltd. All rights reserved.
Keywords: Adaptive control; Robotic manipulators; Nonlinear control; Parameter estimation; Least-squares estimator
1 Introduction
Robot manipulators are nonlinear, multi-input/
multi-output mechanical systems known to have a fairly
well-de"ned dynamic structure; however, the physical
parameters (e.g., payload masses, friction coe $cients,
etc.) used to complete the dynamic description are not
often known precisely. This problem of parametric un-
certainty has led many researchers to propose adaptive
control laws that compensate for the unknown robot
manipulator parameters while still maintaining accurate
link position tracking. References to some representative
work on adaptive robot control can be found in Queiroz
et al. (1996).
Most of the above work employs a ‚yapunov-based
approach in the design of the adaptive controller (i.e., the
controller and the adaptive update law are designed in
conjunction via a single Lyapunov function). Unfortu-
nately, this use of a single Lyapunov function often

Although mainly applied to linear systems, theestima-
tion-based approach to adaptive control, as opposed to
Lyapunov-based or passivity-based designs, allows for
further #exibility in the construction of parameter update
laws (e.g., prediction error-based gradient or nor-

designed). This #exibility is achieved due to the modular
design of the controller and the update law. A modular
estimation-based-type control approach for robot
manipulators was "rst reported by Middleton and
Goodwin (1988). Roughly speaking, in this work, the
adaptive computed torque controller of Craig et al.
(1986) was augmented with additional terms which al-
lowed the closed-loop error system to be written as
a stable, linear, strictly proper, transfer function with the
link position tracking error as the output and a predic-
tion error related term as the input. This input }output
relationship facilitated the dual objectives of link posi-
tion tracking and controller/update law modularity in
the sense that any parameter update law could be used as
long as its design ensured that: (i) the parameter esti-
mates remain bounded, (ii) the prediction error is square
integrable, and (iii) the estimated inertia matrix is posit-
ive-de"nite (PD) (i.e., a projection-type algorithm is re-
quired in the parameter update law). Recent work by
Krstic and Kokotovic (1995) has successfully extended
previous linear estimation-based techniques to a class of
parametric-strict-feedback nonlinear systems by cleverly
exploiting the nonlinear damping argument (Kokotovic,
1992). Unfortunately, the class of systems considered in
Krstic and Kokotovic (1995) does not encompass the

In this paper, we develop an adaptive link position
tracking controller for robot manipulators which
achieves controller/update law modularity. Speci "cally,
we "rst design a torque input control law to achieve
input-to-state stability with the parameter estimation
error treated as a disturbance input to the link position
tracking error dynamics; hence, the torque controller
ensures closed-loop stability for any adaptive update law
design which ensures that the parameter estimates re-
main bounded. We then illustrate how the design of the
torque input control law yields global asymptotic link
position tracking for any adaptive update law design
which also satis "es the following su $cient conditions:
(i) the prediction error is square integrable, and (ii) the
estimated inertia matrix is PD. Hence, in contrast with
the early work presented in Middleton and Goodwin
(1988), closed-loop stability does not require the esti-
mated inertia matrix to be PD; moreover, the proposed
torque input control law does not require the online
calculation of the inverse of the estimated inertia matrix
as is required for the controller presented in Middleton
and Goodwin (1988). To illustrate the advantages of the
modular adaptive control design, the proposed control
law using a standard least-squares estimator is compared
to the well-known adaptive controller of Slotine and Li
(1991) through dynamic simulations and experimental
results.
The paper is organized as follows. Section 2 introduces
the robot manipulator dynamic model along with some
useful properties. In Section 3, we give the problem
statement. Section 4 presents the input-to-state stability
result while Section 5 presents the link position tracking
result. Simulation and experimental results are presented
in Section 6. Section 7 contains the conclusions.
